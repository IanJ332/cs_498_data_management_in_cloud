I implemented the performance analysis using a Python script with the requests library. I used time.time() to capture the start and end timestamps for each HTTP request to calculate the duration in milliseconds. For Part A, the script sends 10 GET/POST requests sequentially and computes the average. For Part B, it uses a loop of 100 iterations where a unique username is generated via the uuid module, registered on the US instance, and immediately queried on the EU instance to detect replication lag.
